get /links

Retrieve a list of links

  • Request

Query Parameters

  • format: (string)

    Override any header-based content negotiation and use the format requested here. The only valid override format is currently json.

    Example:

    format=json
  • filter: (string)

    Filter the returned results using a substring match.

    Example:

    foo
  • offset: (integer)

    Skip over a number of elements by specifying an offset value for the query

    Example:

    20
  • limit: (integer - default: 20)

    Limit the number of elements on the response

    Example:

    400

post /links

Create a new entry, assign it an id, and return that ID back to the client.

  • Request
  • Response

Query Parameters

  • format: (string)

    Override any header-based content negotiation and use the format requested here. The only valid override format is currently json.

    Example:

    format=json
  • url: required (string)

    A URL that should be stored.

    Example:

    http://example.org/
  • caption: (string)

    A caption or display-name for the link.

    Example:

    The infamous example.org example
  • category: (string)

    A comma separated list of categories that this link belongs to.

    Example:

    foo, bar, baz, examples

/links/{linkId}/ get post put delete

An item in the collection of all links

get

Return the link defined by {linkId}.

post

Replace (a very blunt update) the entry specified by {linkId} with the link defined in this request. If an ID is included in the payload it is ignored in favour of {linkId} from the URL.

put

Replace (a very blunt update) the entry specified by {linkId} with the link defined in this request. If an ID is included in the payload it is ignored in favour of {linkId} from the URL.

delete

Delete the entry defined by {linkId}
