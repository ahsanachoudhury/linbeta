LINORA is a Link Organizer and Archiver – it helps you Organise and Archive things you find on the web.

The purpose is to help people who use browsers on multiple computers (tablets, smartphones, etc…) to avoid the need to wearily create much the same bookmarks (or favourites, or whatever the browser call them) on each different browser.

Certain browsers already do bookmark syncing at the cost of a little “setting up”, which is great if you use only that single browser on all computers: but it doesn’t solve the problem if you use different browsers for some reason.

We hope you find Linora useful!

Navigation

Linora has three panels which can be at the top top of the page. The default panel displays Links. The Add panel can be used for adding and editing entries. This is the Helpem> panel.

Links Panel Keyboard Shortcuts

Escape
Clear current search
Enter
Load currently selected entry.
Up Arrow
Select link above current selection.
Down Arrow
Select link below current selection.

Adding a Link

Drag it

  1. Find a link somewhere
  2. Drag and drop it onto the linora screen
  3. Improve the auto-generated caption and tags
  4. Press Add

Type it

  1. Type a link into the URL field
  2. Type a caption / name for the link
  3. Type a comma separated list of tags
  4. Press Add

Editing a link

  1. Click the Edit button next to any entry
  2. Change things
  3. Press Add
